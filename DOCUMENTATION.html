1. Getting Started

1.1 Prerequisites

To run the project successfully, you need to have the following installed:

  • Node.js (Version 18+ recommended).
  • Package manager npm or yarn.

1.2 Setup Instructions

  1. Install Dependencies: Open your terminal in the project directory and run:
    npm install
  2. Run Development Server: To start the local server:
    npm run dev

    The application will be available at: http://localhost:5173/

  3. Production Build: To create the final files ready for hosting:
    npm run build

2. Theme and Color Customization

The theme uses CSS Variables for easy customization. You can control all primary colors from a single file.

2.1 Modifying Primary Colors

Edit the following file:

/src/assets/styles/theme.css

Adjust the following variables:

:root {
    --color-gold: #D4AF37; /* Change this for main branding color */
    --color-background: #F8F8F8; /* Light mode background */
    --color-text: #1E1E1E; /* Primary text color */
}

html[data-theme="dark"] {
    --color-background: #121212; /* Dark mode background */
    --color-text: #F8F8F8; /* Dark mode text color */
    --color-card-background: #1E1E1E; /* Dark card background */
}

2.2 Dark Mode Persistence

Dark Mode state is automatically managed and saved using the Pinia Store and localStorage. No code modification is required for this feature to work across sessions.

3. Google Firebase Integration

The template is pre-configured for Firebase to facilitate data fetching and user authentication.

3.1 Configuration File Setup

To connect the template to your own Firebase project, follow these steps:

  1. Create a new project in the Firebase Console.
  2. In Project Settings, register a new Web App.
  3. Copy the essential Configuration Object data.
  4. Paste the data into the following file:
    /src/firebase/config.ts

The file should look like this after your configuration is added:

const firebaseConfig = {
    apiKey: "YOUR_API_KEY_HERE",
    authDomain: "YOUR_AUTH_DOMAIN_HERE",
    projectId: "YOUR_PROJECT_ID_HERE",
    // ... other details ...
};
